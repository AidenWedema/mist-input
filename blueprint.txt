main window:
- buttons to open the virtual controller menu and input mapper
- virtual controller button opens a new screen displaying all inputs on the connected controller, with options to add/remove them.
- input mapper opens a new screen whit every input in the virtual controller and what it is bound to, with options the clear, change and configure the bound input.
- input display that shows what buttons are pressed and where the joysticks are.
- has a virtual controller class.

virtual controller:
- holds every input of the connected controller.
- extra features like the direction of a joystick in cardinal directions.
- uses the keybind class

keyclass:
- translates inputs from the virtual controller to the mapped inputs on the keyboard and mouse.
